Catastrophe !

Bien que la musique soit revenue, vous vous rendez compte que ce n'est pas le cas de la lumière.

En effet, la boule à facette est verrouillée par un code laissé par le DJ originel : Franck Master Beats.

Sans elle, impossible d'allumer les spots, les strobos, ni de faire un moonwalk sur la piste de danse.

À vous de retrouver les reliques de Franck Master Beats, résoudre leurs énigmes pour ainsi décrypter le code d'activation de la boule.
